# 1️⃣阶段一
## 1.GitHub上不去问题  
1. 原因  
- 校园网通常对境外网站带宽限制更严格，路由绕路多，延迟高、丢包率大  
- 国内普遍存在的 DNS 污染、GFW 干扰，也会导致 GitHub 连接不稳定或无法访问  
- 本地网络环境（如家庭宽带、移动网络）也可能因运营商出口拥堵出现类似问题
2. Watt Toolkit作用
- 本质：它是网络加速工具，并非单纯“提升网速”，而是优化访问境外服务的连接路径与稳定性
## 2.链接问题  
1. 问题：原本Task的内容没有分层，也无链接可点，一点都不清晰，网络上找教程也没找到适合的
2. 解决：多次点开mentor的主页，学习模仿他的界面，并向AI提问，最终做出分层链接

# 2️⃣阶段二  
## 1.输入错误文件名  
   - mv 错误文件名 正确文件名  
   - git commit -m "fix: 修改文件名"
## 2.换行符警告  
   - 不影响功能，系统兼容性提示  
## 3.本地分支上游丢失  
   - git branch --unset -upstream  
   - git checkout main  
   - git merge master  
   - git push origin main  
## 4.同步远程问题  
   - 在Bash或VSCode终端执行：git push origin main  
   - VSCode记得同步，不能光保存本地仓库就以为上传到远程  
## 5.冲突  
   - 打开冲突文件，手动编辑  
   - git add 冲突文件名  
   - git commit -m "fix:解决文件冲突"  
   - git push origin main
